노무사 학습 플래너

=== 서버 주소 ===

https://cpla-production.up.railway.app/index.html

=== 주요 기능 ===

1. 과목별 워드클라우드
   - 1교시: 노동법1, 노동법2
   - 2교시(경영학): 민법, 사회보험법, 경영학
   - "전체" 선택 시 모든 과목 자동 통합

2. 다개년 통합 워드클라우드
   - 최근 3개년 통합
   - 최근 5개년 통합
   - 페이지 연도 기준으로 과거 N년 조회

=== 배포 시 주의사항 ===

Railway 재배포 시 데이터 유지를 위해 Volume 설정 필요!
자세한 내용은 RAILWAY_SETUP.md 참고

=== 로컬 실행 ===

node server.js
http://localhost:3000
