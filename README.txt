노무사 학습 플래너

=== 로컬 실행 방법 ===

1. Node.js 설치 (https://nodejs.org/)

2. 터미널에서 이 폴더로 이동:
   cd "C:\Users\SSAFY\Desktop\노무사"

3. 패키지 설치:
   npm install

4. 서버 실행:
   npm start

5. 브라우저에서 열기:
   http://localhost:3000
